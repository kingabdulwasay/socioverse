# 🌐 SocioVerse – Social Media Web App

SocioVerse is a modern social media web application that allows users to connect, share posts, and interact seamlessly. It includes authentication, media uploads, and session handling for a smooth user experience.

---

## 🔗 Live Demo
🌍 Live Project: https://soocioverse.netlify.app/


## 🚀 Features

### 🔐 Google Authentication
- Secure login using Google Auth
- User profile auto-creation after login
- Persistent authentication state

### 📝 Post Creation
- Create text-based posts
- Upload images with posts
- Instant feed updates

### 💾 User Session Management
- Session stored using **localStorage**
- Auto-login on revisit
- Smooth logout handling

### ☁️ Media Management
- Upload images directly to cloud storage
- Fast image retrieval and rendering
- Optimized media handling for posts

---

## 🛠️ Tech Stack

- HTML, CSS, JavaScript
- Firebase Authentication
- Firebase Firestore / Realtime Database
- Firebase Storage (for images)
- LocalStorage (session handling)
